# DepScan Proto
Protocol Buffers library for the DepScan ecosystem.

## Dependencies
Build dependencies used:
- Rust compiler v1.78.0
- Protobuf compiler v3.2.14

See [Cargo.toml](Cargo.toml) for lib dependencies.
